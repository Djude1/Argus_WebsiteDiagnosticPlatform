## 計畫偏離說明

計畫 Step 3 建議用 `manage.py shell -c "..."` 來 seed，但 PowerShell heredoc 在 Windows cp950 console 下會把繁體中文編碼壞掉（已實測 `unrecognized arguments` 錯誤）。改用 **data migration** 是更穩定的等效解法：

- 同樣達成「seed 常駐法律公告」目的
- get_or_create 保有冪等性
- git-tracked，其他開發者 pull 後 `migrate` 自動套用，不需另外執行命令
- 提供 reverse function 可 rollback
