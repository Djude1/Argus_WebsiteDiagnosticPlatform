# OpenAIDevice_For_VisualImpairment
This is an open-source project that incorporates an open-source artificial intelligence model into an IoT device (ESP32, microphone, camera, etc.) designed specifically for visually impaired people.
這是一個搭載開源人工智慧模型的IoT裝置（ESP32、麥克風、攝影機等），專為視障人士設計的開源專案。

參考來源：https://github.com/AI-FanGe/OpenAIglasses_for_Navigation (AI 智能盲人眼镜系统 🤖👓)
