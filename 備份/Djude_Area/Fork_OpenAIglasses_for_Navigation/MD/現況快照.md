## 已知 Bug（後續修，跟今天 commit 無關）

### Bug: APP 重連後 viewer 累積 → 黑屏

**症狀**：每次「啟動盲道導航」server log `[VIEWER] Browser connected. Total viewers: N` 數字 +1，累積後 APP 端避障畫面卡住、黑屏。APP 重連 server 才恢復。

**根因方向（未確定）**：
1. APP 端 dart 啟動導航時建新 ws/viewer 不關舊（最有可能）
2. 或 navigator state machine 在多 viewer 時某條件下卡住

**workaround**：APP 黑屏時重連 server。

**詳見**：`memory/feedback_viewer_accumulation_blackscreen.md`

**注意**：此 bug 4/9 b0c79f1 架構就有，**跟 5e6c05b commit 完全無關**。
