## 三、 硬體接線與 GPIO 定義

### 1. 攝影機模組 (OV3660 / FPC 排線)
透過底部 FPC 介面連接，接腳定義如下：

| 功能 | GPIO | 說明 |
| :--- | :--- | :--- |
| XCLK | GPIO 10 | 系統時鐘 |
| PCLK | GPIO 13 | 像素時鐘 |
| VSYNC | GPIO 38 | 影格同步 |
| HREF | GPIO 47 | 行同步 |
| SIOD (SDA) | GPIO 40 | I2C 資料 |
| SIOC (SCL) | GPIO 39 | I2C 時鐘 |

**影像資料線 (D0 ~ D7):**
* Y2: GPIO 15 | Y3: GPIO 17 | Y4: GPIO 18 | Y5: GPIO 16
* Y6: GPIO 14 | Y7: GPIO 12 | Y8: GPIO 11 | Y9: GPIO 48

### 2. IMU 陀螺儀 (ICM42688P - SPI 模式)
| ICM42688P | XIAO 腳位 | GPIO | 程式碼定義 |
| :--- | :--- | :--- | :--- |
| SCLK | D0 | GPIO 1 | IMU_SPI_SCK |
| MOSI | D1 | GPIO 2 | IMU_SPI_MOSI |
| MISO | D2 | GPIO 3 | IMU_SPI_MISO |
| CS | D3 | GPIO 4 | IMU_SPI_CS |
| VCC | 3.3V | - | 電源 |
| GND | GND | - | 共地 |

### 3. 音訊輸出 (MAX98357 + 喇叭)
| MAX98357 | XIAO 腳位 | GPIO | 程式定義 |
| :--- | :--- | :--- | :--- |
| Vin | 5V | - | 電源 |
| GND | GND | - | 共地 |
| BCLK | D8 | GPIO 7 | I2S_SPK_BCLK |
| LRC | D9 | GPIO 8 | I2S_SPK_LRCK |
| DIN | D10 | GPIO 9 | I2S_SPK_DIN |
| GAIN | GND | - | 預設增益 |
| SD | 5V | - | 啟用模式 |

### 4. 板載 PDM 麥克風 (Sense 擴展板內建)
* **Clock:** GPIO 42
* **Data:** GPIO 41
